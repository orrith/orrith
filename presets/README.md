# Community Presets

Share your aesthetic preset here.

## How to contribute

1. Create `your-preset-name.css` in this directory
2. Use `body[data-preset="your-preset-name"]` as the root selector
3. Override CSS variables defined in `src/public/style.css` `:root` (see existing presets in `src/public/style.css` for reference)
4. Add a screenshot at `presets/screenshots/your-preset-name.png` (1280x720 recommended)
5. Open a PR — we'll review and merge

## Existing presets (built-in)

The 5 built-in presets ship in `src/public/style.css`:

- **pixel** — 8-bit retro · Press Start 2P · pink shadow
- **visor** — HMD cockpit · cyber · cyan accent
- **monolith** — quiet · monospace · zero ornament
- **doodle** — handwritten notebook · cream paper
- **minimal** — whitespace-led · Inter · modern simple

## What gets accepted

- Distinctive aesthetic (not a slight variation of an existing preset)
- All 4 tabs (ROADMAP / METRICS / STATUS / PREVIEW) look polished
- No external image dependencies (web fonts via Google Fonts OK)
- License: MIT (so others can build on it)
