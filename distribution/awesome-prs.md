## PR submission template

```md
**Source code**: https://github.com/orrith/orrith
**Description**: The ops cockpit for solo builders — Markdown ops + multi-source metrics in one HUD with 5 aesthetic presets.
**Why it fits this list**: <category-specific reason>
**License**: MIT
**Active**: yes (1st release in 2026-MM-DD)
```

## Submission etiquette

- Read CONTRIBUTING.md for each list before opening
- Don't spam multiple lists same day; spread over 1-2 weeks
- If a list has strict criteria (X stars / Y stable releases), wait until orrith meets them
- Be polite if rejected; "you're not a fit" feedback is data
